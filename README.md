## 🛠️ Local Setup Instructions

```bash
# 1. Clone or open repository folder
cd my_portfolio

# 2. Install dependencies
npm install

# 3. Start local development server
npm run dev
```

Open `http://localhost:3000` in your browser.

## 📤 Pushing Source Code to GitHub

```bash
# 1. Create a new repository on GitHub (e.g. named "aakash-flutter-portfolio")

git remote add origin https://github.com/<your-github-username>/aakash-flutter-portfolio.git

git branch -M main
```

## 🌐 Live Deployment Options

### Option A: Deploying to GitHub Pages (Automated)
```bash
npm run deploy
```
*Your site will be live at `https://<your-github-username>.github.io/aakash-flutter-portfolio/`*

### Option B: Deploying to Vercel (Recommended)
1. Go to [vercel.com](https://vercel.com) and log in with GitHub.
2. Click **"Add New Project"** and import `aakash-flutter-portfolio`.
3. Keep default settings (`Framework Preset: Vite`) and click **"Deploy"**.
4. Your site will be live in 30 seconds with a free HTTPS link!
